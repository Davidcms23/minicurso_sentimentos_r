# minicurso_sentimentos_r
Este repositório contém o material utilizado no minicurso "Introdução à Análise de Dados com Python", oferecido durante a Semana da Estatística da UFRN 2024. O objetivo do minicurso é apresentar uma introdução a análise de sentimentos utilizando o R, utilizando léxicos como o OpLexicon e o NRC.
